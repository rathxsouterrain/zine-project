Quick Start Guide

Selamat datang di Shopee Flash Sale Bot!

Ikuti langkah-langkah berikut untuk memulai:

  1. Login Session - Buka "Session Manager" dan tambahkan cookies dari akun Shopee Anda
  2. Setting Produk - Masukkan URL produk target di "Configuration"
  3. Jalankan Bot - Klik "START BOT" di "Bot Control"
  4. Monitor - Pantau progress di dashboard dan logs
1. Session
2. Config
3. Start Bot
Perhatian: Bot ini menggunakan automation browser yang mungkin terdeteksi oleh sistem anti-bot Shopee. Gunakan dengan bijak dan risiko ditanggung sendiri.
